# 📗 Intermediate Python Books

Welcome to the **Intermediate Python Books** section!

These books are for learners who:
- Already understand Python basics (variables, loops, functions)
- Want to explore deeper concepts like data structures, object-oriented programming, APIs, and more
- Are building small projects or solving real-world problems with Python

---

## 📚 Books in This Folder

### 1. [Think Python: How to Think Like a Computer Scientist](https://greenteapress.com/wp/think-python-2e/)
**Author:** Allen B. Downey  
A great transition from beginner to intermediate. Teaches you how to think logically, write clean code, and build better programs.

---

### 2. [The Hitchhiker’s Guide to Python](https://docs.python-guide.org/)
**Authors:** Kenneth Reitz & Contributors  
A highly practical book for writing professional Python code. Covers tools, project structure, virtual environments, packaging, and more.

---

### 3. [Python for Everybody](https://do1.dr-chuck.com/pythonlearn/EN_us/pythonlearn.pdf) *(Also suitable for beginners)*  
**Author:** Dr. Charles Severance  
If you've finished the basics, this book will help you apply your Python knowledge to work with data and real-world tasks.

---

### 4. [The Recursive Book of Recursion](https://inventwithpython.com/recursion/)
**Author:** Al Sweigart  
Teaches recursion — a tricky but powerful concept — in a beginner-friendly way using Python and fun examples.
